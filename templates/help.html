Length of ID: Number of characters for each unique ID (e.g. 8). Leave blank to use full UUID.

Number of Links: Total links/IDs to generate. This will auto-fill when an ID file is uploaded.

Survey Link: The base URL for appending unique IDs. (e.g., https://survey.com/?id=)

Prefix: Optional prefix added to each ID. Useful for wave or batch tagging.

Include &ptest=0: Appends &ptest=0 to all links if checked.

Upload ID File: Accepts `.txt` or `.xlsx` files with one ID per row. Overrides auto-generated IDs.

Generate Test Links: If checked, generates test links (TEST_1, TEST_2, ...) in a separate file.

Outputs:

  • .dat file with the list of IDs
  • .xlsx file with the links
  • TestLinks file (if test link generation is enabled)
